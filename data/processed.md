# Processed data

This folder contains the normalized version of the logic-based dataset.

## What is processed

- File: data/processed/logic_based.json
- Source: data/logic_based.json
- Generated by: data/preprocess_logic.ipynb

## What changes are applied

- Normalizes quantifiers to `ForAll(x, ...)` / `Exists(x, ...)`.
- Replaces Unicode operators with ASCII tokens (e.g., `→` to `->`, `∧` to `AND`).
- Replaces membership symbol `∈` with `IN`.
- Ensures `NOT` is separated from parentheses (e.g., `NOT (`).
- Closes unmatched parentheses to keep parser compatibility.
- Preserves the original JSON structure and non-FOL fields.

## How to regenerate

Run the notebook data/preprocess_logic.ipynb from the data/ folder so paths resolve correctly.

## Physics preprocessing

- File: data/physic.csv
- Notebook: data/preprocess_physics.ipynb
- Steps:
	- Collects unique characters before and after normalization.
	- Replaces non-ASCII scientific notation (e.g., 1.00 × 10⁷) with ASCII equivalents.
	- Normalizes multiplication to `*` with spacing (e.g., 1.82 * 10^-3).
	- Wraps powers as `10^{...}` (e.g., 10^-4 to 10^{-4}).
