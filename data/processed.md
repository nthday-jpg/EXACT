# Processed data

This folder contains the normalized version of the logic-based dataset.

## What is processed

- File: data/processed/logic_based.json
- Source: data/logic_based.json
- Generated by: data/process.ipynb

## What changes are applied

- Normalizes quantifiers to `ForAll(x, ...)` / `Exists(x, ...)`.
- Replaces Unicode operators with ASCII tokens (e.g., `→` to `->`, `∧` to `AND`).
- Replaces membership symbol `∈` with `IN`.
- Preserves the original JSON structure and non-FOL fields.

## How to regenerate

Run the notebook data/process.ipynb from the data/ folder so paths resolve correctly.
