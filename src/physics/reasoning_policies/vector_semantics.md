##1. VECTOR INVARIANT
Force and field are vectors.

Never directly add magnitudes unless:
- vectors are collinear
AND
- directions resolved

Otherwise:
- resolve components
- sum components
- compute magnitude afterward

##2. VECTOR VS SCALAR
Signed scalar addition allowed only for:
- collinear vectors

##3. DISPLACEMENT VECTOR
d_vec = target_position - source_position

Always:
source → target

##4. COMPONENT SUM
Fx_total = ΣFx
Fy_total = ΣFy

Magnitude:
sqrt(Fx_total**2 + Fy_total**2)

##5. CANCELLATION
Equal magnitudes alone do NOT imply cancellation.

Cancellation requires opposite directions.
