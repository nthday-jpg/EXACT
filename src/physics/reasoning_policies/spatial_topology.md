## 1. COLLINEAR DETECTION
If:
r1 + r2 == AB
→ point lies on segment AB

If:
|r1 - r2| == AB
→ point lies outside segment AB

## 2. PERPENDICULAR BISECTOR
If:
PA == PB
→ perpendicular bisector candidate

## 3. TRIANGLE VALIDITY
Check:
a + b > c

Reject invalid triangles.

## 4. MIDPOINT
If midpoint:
M = (A + B)/2

## 5. SYMMETRY
Equal magnitudes do NOT imply cancellation.

Cancellation requires opposite directions.
