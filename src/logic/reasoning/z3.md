# Z3 utilities

Helpers for parsing simple FOL-like strings into Z3 expressions and verifying entailment.

## Files

- verifier.py: Lightweight parser, symbol manager, and verifier utilities to parse FOL formulas and verify logical entailment via the Z3 SMT solver.

## Supported input

- Quantifiers: `ForAll(x, ...)`, `Exists(x, ...)`
- Boolean operators: `AND`, `OR`, `NOT`, `->`, `<->`
- Predicates: `P(x)` / `R(a, b)`
- Comparisons: `=`, `!=`, `>=`, `<=`, `>`, `<`
- Membership: `x IN y` (parsed as predicate `In(x, y)`)
- Literals: integers, floats, and single-quoted strings

## Main entry points

- `parse_formulas(formulas, sort_name="U")` returns:
	- `symbols`: shared symbol table (`Z3Symbols`)
	- `formula_exprs`: list of Z3 Bool expressions representing the parsed formulas
- `verify_with_z3(premises_fol, conclusion_fol)` returns:
	- `result`: verification check result (`sat`, `unsat`, or `unknown`)
	- `proof`: Z3 expression representing the proof (if `unsat`)
	- `unsat_core`: minimal core tracking variables that contributed to the proof (if `unsat`)
	- `model`: counterexample Z3 Model (if `sat`)

## Example

```python
from src.logic.reasoning.verifier import parse_formulas, verify_with_z3
from z3 import Solver, sat, unsat

# --- Option 1: Parse and solve manually ---
premises = ["Human(Socrates)", "ForAll(x, (Human(x) -> Mortal(x)))"]
symbols, exprs = parse_formulas(premises)

s = Solver()
s.add(*exprs)

if s.check() == sat:
    print("Satisfiable!")
    print(s.model())

# --- Option 2: Verify entailment via verify_with_z3 ---
verification = verify_with_z3(
    premises_fol=["Human(Socrates)", "ForAll(x, (Human(x) -> Mortal(x)))"],
    conclusion_fol="Mortal(Socrates)"
)

if verification["result"] == unsat:
    print("Proven! Socrates is mortal.")
    print("Proof:", verification["proof"])
```

## Notes

- This parser is intentionally minimal and does not implement full FOL or rich arithmetic.
- Use the logical normalization utilities before parsing to translate custom symbols (like "¬", "∧", "∨", "→", "↔") to parser-safe terms.
