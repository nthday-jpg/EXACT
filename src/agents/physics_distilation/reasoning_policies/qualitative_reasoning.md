Prioritize direction of change, conservation laws, limiting behavior, and resonance implications.

## 3. OUTPUT STYLE (TEXT ONLY)
Never force numerical or symbolic formula evaluations. The 'ans' variable inside 'python_code' must be a raw text description string enclosed in a list.

- Example for trends: ans = ["increases"]; unit = ["-"]
